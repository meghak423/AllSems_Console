# Agile Project Management - Sales Call Assistant

## Project Vision
To provide real-time, AI-driven guidance for sales professionals to handle objections and improve conversion rates during live calls.

---

## Sprint 1: Foundation & STT Integration
**Goal:** Establish the real-time transcription pipeline.
* **User Story:** As a Sales Rep, I want to see my speech converted to text instantly so I can track the conversation.
* **Tasks:**
    * Setup Python environment and API keys (OpenAI/AssemblyAI).
    * Implement real-time microphone stream processing.
    * **Status:** Completed

## Sprint 2: AI Strategy Engine
**Goal:** Implement intelligence for objection handling.
* **User Story:** As a Sales Rep, I want the AI to suggest responses when a customer mentions "Price" or "Budget."
* **Tasks:**
    * Create prompt templates for sales coaching.
    * Integrate LLM to analyze transcript context.
    * **Status:** Completed

## Sprint 3: UI/UX & Sentiment Analysis
**Goal:** Create a professional dashboard for the end-user.
* **User Story:** As a user, I want a clean interface that shows customer sentiment (Positive/Negative).
* **Tasks:**
    * Build Streamlit Dashboard.
    * Add visual indicators for "Sentiment Score."
    * **Status:** Completed
