1. Farthest Indices 
Given an array A of N positive integers. Print the maximum of j - i subjected to the constraint of A[i] ≤ A[j].

Input Format
The first line of input contains an integer N. The second line of input contains an array of size N.

Output Format
For the given input, print a single line representing the answer.

Constraints
1 ≤ N ≤ 103
0 ≤ Ai ≤ 109

Example
Input
9
10 9 10 31 12 90 320 313 9 

Output
7

Explanation

If we choose the element at index 0 and the element at index 7, where 10 <= 313, the difference between their indices (j - i) is 7, which is the maximum value we can obtain. If we choose the element at index 1 and the element at index 8, where 9 <= 9, even for these indices, j - i equals 7.

o/p  : # Enter your code here. Read input from STDIN. Print output to STDOUT

n = int(input())
arr = list(map(int,input().split()))
max_dist = 0

for i in range(n):
    for j in range(n-1,i,-1):
        if arr[i] <= arr[j]:
            max_dist = max(max_dist,j-i)
            break

print(max_dist)

        r = n//2

Example
Input
5
8
16808
9
23
97158

Output
4
16

2
8

Explanation

Self Explanatory


o/p : import java.io.*;
import java.util.*;

public class Main {

    public static void main(String[] args) {
        /* Enter your code here. Read input from STDIN. Print output to STDOUT. Your class should be named Main. */
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();

        while(T --> 0) {
            int count = 0;
            long n = sc.nextLong();
            long sqrtn = (long)Math.sqrt(n);

            for (long i=1;i<=sqrtn;i++) {
                if ( n % i == 0) {
                    count++;
                    if( i != n/i) {
                        count++;
                    }
                }
            }
            System.out.println(count);
        }
        sc.close();   
    }
}
